HOLES - BEARING HOLE EXAMPLE (1.125" diameter)
────────────────────────────────────────────────────────────────────────

WITHOUT COMPENSATION (WRONG!):

CAD: 1.125" diameter hole (0.5625" radius)
Tool: 4mm (0.157" diameter, 0.0785" radius)

       Tool CENTER follows CAD circle
              ╱─────────╲
            ╱     ●       ╲  ← 0.5625" radius path
          ╱      ╱ ╲       ╲
         │      │   │       │
         │       ╲─╱        │ Tool radius: 0.0785"
          ╲               ╱
            ╲___________╱

Final hole diameter: 1.125" + 2×0.0785" = 1.282" (TOO BIG!)

────────────────────────────────────────────────────────────────────────

WITH COMPENSATION (CORRECT!):

Toolpath radius: 0.5625" - 0.0785" = 0.4840"

       Tool CENTER follows smaller circle
              ╱─────────╲
            ╱      ●      ╲  ← 0.4840" radius path
          ╱      ╱   ╲      ╲   (compensated)
         │      │  ●  │      │  
         │       ╲   ╱       │ Tool cuts to exact
          ╲      ╲─╱        ╱  1.125" diameter
            ╲___╱   ╲___╱
              ╲_______╱
           1.125" final hole ✓

KEY TAKEAWAYS:
────────────────────────────────────────────────────────────────────────
✓ Perimeter: Tool path OUTSIDE the part (offset OUTWARD)
✓ Pockets:   Tool path INSIDE the pocket (offset INWARD)
✓ Holes:     Tool path is SMALLER circle (reduced radius)

Without compensation: Parts are WRONG SIZE
With compensation:    Parts match CAD EXACTLY

ALWAYS specify --tool-diameter for correct results!
